# administratie-app
Administratie app (hosted on Heroku)

## What is this?
Express/React application

## How to deploy?
There are several ways

### Deploy to Heroku
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

### Clone and deploy yourself
1. Close repo
2. run `npm install`
3. run `npm run build`
